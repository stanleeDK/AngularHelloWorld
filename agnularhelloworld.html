Intro to Angular

  • Angular lets you do MVC in a client server setup
  • It also allow you to do data binding. Basically you define your model and then bind it to html elements. You can do server to client using {{}} or viceversa using directives and html tag to model attribute binding.
Here the html tag has ng-app which tells angular that the entire html app is under it's purview. You also use the ng-model directive to bind an html element to the data model.
You also used the ng-bind directive.
{{1 + 2}}


With Angular it takes care of the data binding. You've defined the model, attached html tags to it. It handles the rest. No need to write event handlers, find DOM elements and send them.
